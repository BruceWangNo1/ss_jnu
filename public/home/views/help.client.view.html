Getting help

A detailed instruction manual.

Download

To start to use our service, the first thing you need to do is to download the client software suitable for you.

sudo add-apt-repository ppa:hzwhuang/ss-qt5
sudo apt-get update
sudo apt-get install shadowsocks-qt5

Mac Configuration

  1. Install and open the software.

  2. Click the paper plane sign on the menu bar.

  3. Go to 'Servers' -> 'Open Server Preferences'.

  4. Click '+' sign and start to add configuration information.

  5. 'Address' is in 'Control Panel' -> 'Node List' -> 'IP' from SS-JNU. Immediately follows 'Address' is the port number which can be found at 'Control Panel' -> 'Configuration Information'. There would not be any problem for the following two blanks and you can leave 'Remark' blank. Click 'OK' to finish.

  6. Click the paper plane on the menu bar. Click 'Global Mode' if you want all your traffic to go through shadowsocks and 'Auto Proxy Mode' (which is prefered) if you only want certain blocked traffic to go through shadowsocks. Then click 'Turn Shadowsocks On'.

  7. Open your Safari and start to enjoy the free Internet.

Note: Close shadowsocks when you are not using services like Google or Youtube to have better bandwidth.

Windows Configuration

  1. Most of the configuration steps follows Mac.

  2. If you think you have configured shadowsocks correctly but you cannot open google, you may need to configure you web brower. Here is an instruction for Chrome or 360 Safe Browser.

    1. Download SwitchyOmega

    2. Drag the SwitchyOmega copy you downloaded to your Chrome browser and install it.

    3. Click the SwithyOmega sign on Chrome's menu bar and click 'Options'.

    4. Click 'New Profile'.

    5. Configure the new profile following the example in the picture.

    6. Click 'Apply changes'

    7. Click 'ss_jnu_socks5' to use the proxy.

    8. Enjoy the free Internet!

  3. Choose 'System Proxy' rather than 'ss_jnu_socks5' and shadowsocks when you are not using services like Google or Youtube to have better bandwidth.
